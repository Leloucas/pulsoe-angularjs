# Sitio Web Pulso

Diseño del front-end para el sitio web PulsoE, bolsa de trabajo en Hermosillo, Sonora.


### Instalación

Se colocan los archivos en la carpta raís del servidor y se corre el siguiente comando:

```
bower install
```
